# Portfólio Profissional Web

Projeto desenvolvido para a disciplina de **UI/UX** com o objetivo de criar um portfólio profissional web utilizando frameworks modernos de front-end.

O site apresenta informações sobre a equipe, habilidades técnicas, projetos desenvolvidos e formas de contato, com foco em **design visual, organização de interface e responsividade**.

---

## Tecnologias Utilizadas

Este projeto foi desenvolvido utilizando as seguintes tecnologias:

- HTML5  
- CSS3  
- Bootstrap  
- Bootstrap Icons  

O framework Bootstrap foi utilizado para garantir:

- Layout responsivo  
- Grid system  
- Componentes visuais modernos  
- Melhor organização da interface

## Funcionalidades

O portfólio apresenta as seguintes seções:

### Apresentação
Introdução ao projeto e apresentação geral do portfólio.

### Projetos
Seção destinada a apresentar trabalhos ou projetos realizados.

### Habilidades
Lista de tecnologias e linguagens utilizadas pela equipe, incluindo:

- HTML  
- CSS  
- Bootstrap  
- JavaScript  
- Python  
- C#  
- SQL  

### Equipe
Apresentação dos integrantes do grupo com:

- foto de perfil  
- links para GitHub e LinkedIn  
- botão **Ver Perfil** com modal contendo mais informações

### Contato
Formulário visual para contato.

## Responsividade

O site foi projetado para funcionar corretamente em diferentes tamanhos de tela:

- Desktop  
- Tablet  
- Smartphone  

Isso foi possível graças ao sistema de grid do Bootstrap.
